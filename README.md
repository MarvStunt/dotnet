## Particularités

**Si le serveur de jeu est sur une autre machine :**
**Modifier l'URL du serveur :**
   - Dans l'éditeur Godot, ouvrir le fichier `NetworkManager.cs`
   - Localiser la ligne 10 :
   ```csharp
   private string serverUrl = "ws://localhost:5000/gamehub";
   ```
   - Modifier l'adresse
